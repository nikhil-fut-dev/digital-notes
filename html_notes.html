What is HTML?

HTML (HyperText Markup Language) is used to create webpages. It describes the structure of a web page using elements called tags.

Basic Structure of HTML Page

<!DOCTYPE html>
<html>
  <head>
    <title> Title</title>
  </head>
  <body>
    Content goes here...
  </body>
</html>

HTML Headings

Headings are defined from <h1> to <h6>.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>

HTML Paragraph

The <p> tag defines a paragraph.

<p>This is a paragraph.</p>

HTML Links

Links are created using the <a> tag and href attribute.

<a href="https://www.w3schools.com">Visit W3Schools</a>

HTML Images

  • src → image path
  • alt → alternate text
  • width & height → image size
<img src="image.jpg" alt="Sample Image" width="200">

Text Formatting Tags

  • <b> – Bold
  • <i> – Italic
  • <mark> – Highlight
  • <small> – Small text
  • <sub> – Subscript
  • <sup> – Superscript

HTML Comments

Comments are not shown in the browser.
